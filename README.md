# pc-network-speed-monitor
realtime speed monitor via 16x2 LED display using arduino nano

Arduino Nano  -> 16x2 LCD Display (Standard Parallel Interface)
--------------------------------------------------------------
GND           -> GND

5V            -> VCC

D7            -> RS

GND           -> RW 

D6            -> EN

D5            -> D4

D4            -> D5

D3            -> D6

D2            -> D7

5V            -> A (Backlight Anode)

GND           -> K (Backlight Cathode)
